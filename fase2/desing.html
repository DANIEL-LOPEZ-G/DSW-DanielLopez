Proposito del sitio

Este sitio se creo con el animo de explicar la metodologia con la que se maneja el interfaz de usuario
o interaccion humano ordenador por medio de los dispositivos de entrada, salida y mixtos, tambien llamados
perifericos,es por medio de este hadware que el usuario tiene interaccion con un ordenador parapoder ingresar
informacion y asi mismo poder sacar informacion.

Interfaz de Ususario

La interfaz de usuario o también llamada user interface (UI), son todos los elementos con los cuales se crea
un vinculo de interacción de un ser humano con un ordenador, para que por medio de esta se pueda tener una
comunicación más sencilla y eficiente, IngenioVirtual (2021) nos dice " Sea quien sea el usuario o consumidor,
está claro que, cuanto más intuitivo y accesible sea el diseño, mucho mejor", es así que encontramos los
siguientes modelos de interfaz de usuario más conocidos:

• Interfaz de línea de comandos (CLI)
• Interfaz de usuario de texto (TUI)
• Interfaz gráfica de usuario (GUI)
• Interfaz de usuario de voz (VUI)
• Interfaz de usuario natural (NUI)

Interaccion Humano Ordenador

Es la disciplina que estudia la interacción que se crea entre persona y computadora, el desarrollo que se debe
tener en cuanto al software, esta la podríamos llamar también interacción Humano computadora, dentro de la cual
se pueden observar 3 elementos, como lo encontramos Breton(2019) nos dice "HCI consta de tres componentes: los
usuarios, los ordenadores y la interacción entre ellos", en las cuales se pueden encontrar muchas características
como lo son procesadores de texto, juegos y contabilidad.
